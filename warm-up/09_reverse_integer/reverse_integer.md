### Reverse Integer

Given a signed 32-bit integer x, return x with its digits reversed. If reversing x causes the value to go outside the 32-bit integer ranfe [-2^31, 2^31 - 1], then return 0.

Assume the environment does not allow you to store 64-bit integers (signed or unsigned).

**Example 1:**

> Input: x = 123
>
> Output: 321

**Example 2:**

> Input: x = -123
>
> Output: -321

**Example 3:**

> Input: x = 120
>
> Output: 21

**Example 4:**

> Input: 1534236469
>
> Output: 0
>
> Explanation: Because the reverse of 1534236469 is 9646324351 and its out of 32-bit signed integer range. Therefore return 0
