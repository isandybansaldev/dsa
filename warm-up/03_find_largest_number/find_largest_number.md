### Find Largest Number

Write a function that returns the largest number from an array

**Example 1:**

> Input: nums = [2,-1,6,9,4,0]
>
> Ouput: 9
>
> Explanation: Because 9 is the largest number in nums array.

**Example 2:**

> Input: nums = [-3,-2,-8,-9,-7,-1]
>
> Ouput: -1
>
> Explanation: Because -1 is the largest number in nums array.
